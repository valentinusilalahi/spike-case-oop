# SPIKE CASE OOP

* Diskon 

   Perhitungan Promo : 

     ``` 
     ~ New Year discount 5%
     ~ Lebaran discount 10%
     ~ Harbolnas discount 20%
     ```

   Sample :
     
     ```
     Amount : 50000.00
     Kode Promo : New Year
     ```
   
     Result : 
     ```
     Discount : 5%
     Amount yang harus dibayar : 45000.00
     ```

* Parkir

   ```
   spesifikasi :
   
   a. Gedung parkir memiliki lot parkir untuk motor, mobil dan bus.
   b. Motor dapat parkir di lot mana saja,
   c. Mobil dapat parkir di lot mobil atau lot bus.
   d. Bus hanya dapat parkir di lot Bus.
   ```
   
   Sample :
   
   ```
   Jenis Kendaraan : motor
   ```
   
   Result :
   ```
   Lot Parkir yang diperbolehkan : 
   - Area Parkir motor
   - Area Parkir mobil
   - Area Parkir bus 
   ```
